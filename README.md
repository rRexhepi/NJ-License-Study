# NJ Driver's License Study

A focused, no-friction study app for the New Jersey MVC knowledge test. Built for one person — open it on a phone or laptop, study, take the mock test. No accounts, no API keys, no setup.

## What's inside

- **12 chapters** covering everything on the NJ knowledge test:
  GDL system, right of way, speed limits, signs, signals, alcohol & drugs, sharing the road, parking, defensive driving, adverse conditions, emergencies, insurance & penalties.
- **42 flashcards** with a Leitner box system — mark cards as Need practice / Getting there / Mastered as you go.
- **60-question practice quiz** with instant feedback and explanations on every answer.
- **Mock test mode** — 50 random questions, 80% to pass, just like the real thing. Tracks personal best.
- **Brain dump** — a scratchpad that auto-saves for things you want to remember.
- **Dark mode**, mobile-friendly, and progress saved in the browser.

## How to use

Open `index.html` in any browser, or host it as a static site (Netlify, Vercel, GitHub Pages, Cloudflare Pages — all free).

Quickest path to phone + laptop access:

1. Go to [app.netlify.com/drop](https://app.netlify.com/drop).
2. Drag the `nj-license-study` folder onto the page.
3. Netlify gives you a URL like `nj-license.netlify.app`. Open it on phone and laptop.

## Study order that works

1. **Skim the Home page tiles** to see the topic landscape.
2. **Read the Study Guide** straight through, opening each chapter.
3. **Run the flashcards** front-to-back, marking weak ones for repeat passes.
4. **Take the practice quiz** — focus on questions you got wrong.
5. **Take the mock test** when you feel ready. Score 90%+ twice in a row → schedule the real test.

## Heads up

This is a study supplement, not the official manual. Always double-check anything critical (penalties, exact suspension lengths, current rules) against the [NJ MVC Driver Manual](https://www.nj.gov/mvc/license/driver-manual.shtml). Laws change.

## Privacy

Everything stays in the browser's `localStorage`. No analytics, no tracking, no servers. Clearing site data resets progress.
